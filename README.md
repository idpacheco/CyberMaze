# CyberMaze
CyberMaze es un proyecto desarrollado en el marco de la asignatura Estructura de Datos 2 (NRC: 2472)de la Universidad del Norte, bajo la orientación de la profesora Katherine Ospino Bermejo. El juego busca fomentar la importancia de la ciberseguridad mediante retos y minijuegos diseñados para enseñar conceptos esenciales de forma práctica y dinámica. Su objetivo es que el jugador comprenda riesgos digitales comunes y desarrolle criterios básicos para una navegación segura mientras avanza a través de escenarios interactivos y progresivos.
