# CyberMaze – Proyecto Final
Este repositorio presenta CyberMaze, un proyecto desarrollado en la Universidad del Norte para la asignatura Estructura de Datos 2 (NRC: 2472), orientada por la profesora Katherine Ospino Bermejo. CyberMaze es una propuesta educativa cuyo objetivo es promover la importancia de la ciberseguridad y enseñar sus principios fundamentales mediante minijuegos y desafíos interactivos. El repositorio reúne la integración de los dos laboratorios realizados durante el semestre, los cuales pueden consultarse en los siguientes enlaces: Laboratorio 1, disponible en [https://github.com/idpacheco/Laboratorio-1--EDD2], y Laboratorio 2, ubicado en [https://github.com/idpacheco/Laboratorio-2--EDD2]. El desarrollo fue realizado por el grupo: Sahory Blanco Durán (200209154),Juan Diaz Granados Amaya (200207273), Danna Galavis Herrera (200148257) e Isis Pacheco Bravo (200210177).
